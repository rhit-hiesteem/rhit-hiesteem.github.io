Platformer Game

This project is a platformer game coded in Java with collision detection, enemies, item collection, increasingly difficult levels, and more. The project was completed as a final project for the Rose-Hulman CSSE220 course and demonstrates object-oriented design. Players can navigate through multiple challenging levels while collecting various power-ups and avoiding obstacles.

Technologies Used: Java

Verb Vault

This project is a Library Management Website made as a senior project in high school. you search the website, using query searches like "Search By Author", "Search By Title", etc, to browse the catalog of books from an online json database. Verb Vault lets you save books to your read list or "Check Out" books, both of which are saved the next time you go to the website.

Technologies Used: PHP, HTML, CSS, Javascript, SQL
